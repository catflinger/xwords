To open a Guardian Quiptic puzzle:

  1. Go to The Guardian website and find the quiptic puzzle you want.
  2. Find and click the link named Print.
  3. The puzzle will open as a simple text and grid image. Save this page as a PDF file on your computer (see below).
  4. Once you have downloaded the PDF file click the Open a PDF button below and follow the instructions.

NOTE on saving a web page as a PDF file

How you save a web page as a PDF file depends on your browser and operating system.

Using Opera browser choose the Page menu item then select Save as PDF...

Using Chrome or Edge on Windows 11 choose Print menu option (Ctrl+P) and choose Microsoft Print to PDF as the destination printer

For other browsers and operating systems you can read the help pages to find out more
