To open a puzzle in a PDF file you must first copy or download the PDF to a file on your computer. If the PDF is laid out in fairly typical way, say that used in the Guardian for the FT, then the app should be able to automatically extract both the grid and clues form the text.

Alternatively, you can create the gid yourself and enter the clues as plain text.
