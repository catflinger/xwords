Guardian and Azed Puzzles

I am curretnly experiencing a problem downloading puzzles automatically from the Guardian Media websites. This includes the Guardian Cryptic, Azed and Everymam. Until I can find out why requests to the Guardian websites are being refused you will need to download these puzzles manually.

The steps to do this are:

  • Go to The Guardian website and find the puzzle you want
  • Find and click the link named PDF Version
  • In your PDF viewer save (or download) the file to your local hard drive
  • Once you have downloaded the PDF file go to the Open PDF page and follow the instructions
I want to load a puzzle from a PDF file on my computer

NOTE: a better way to get a Guardian PDF is to click the Print link rather then the PDF Version link. If you do this then you will need to save the print page as a PDF using your browser's built-in PDF converter.

The advantage of this method is that Print version of the puzzle does not contain the previous week's solutions and various other extraneous text that is present in the normal PDF download. The presence of this extra text makes it hard for my app to distinguish between the real clues and the other text.

  • Go to The Guardian website and find the puzzle you want
  • Find and click the link named Print
  • Using your browser's PDF converter save the print version as a PDF file on your computer
