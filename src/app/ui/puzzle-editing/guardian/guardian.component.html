The steps to do this are:

  1. Go to The Guardian website and find the puzzle you want
  2. Find and click the link named PDF Version
  3. In your PDF viewer save (or download) the file to your local hard drive
  4. Once you have downloaded the PDF file go to the Open a PDF and follow the instructions
