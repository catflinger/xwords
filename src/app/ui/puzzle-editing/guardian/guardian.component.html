To open a Guardian/Observer puzzle:

  1. Go to The Guardian website and find the puzzle you want
  2. Find and click the link named PDF Version
  3. In your PDF viewer save (or download) the file to your local hard drive
  4. Once you have downloaded the PDF file click the Open a PDF button below and follow the instructions

NOTE on Azed

On the Azed web page the link to show a PDF is confusingly named Click here to access print version. Don't panic, this link does exactly the same thing as the PDF Version link on the other crosswords.

NOTE on Quiptic

For some reason the Guardian Quiptic does not have a PDF link at all.

To get a PDF for the Quiptic you will need to follow the Print link and then save/download the resulting page as a PDF using your browser's built-in PDF tool. How you do this depends on your browser.

Using Opera browser choose the Page then Save as PDF... menu options

Using Chrome on Windows 11 choose Print then select Microsoft Print to PDF as the destination

For other browsers and operating systems you can read their help pages to find out more
